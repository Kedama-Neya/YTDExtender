YouTube の動画の説明を自動的に展開する Google Chrome 向け拡張機能です。

実際には、SPA や SSR、SSG、JavaScript のほか、デバッグ環境下でのみバグまみれになる ServiceWorker も
何もかも理解できなかったので、萎えて思考放棄して毎秒チェックして動画説明を力技でこじ開けることにした雑な拡張機能です。
試しに動画説明を閉じてみてください。通常であれば 1 秒以内でまた開きます。

-----

導入方法
1. chrome://extensions を開く。
2. 右上のデベロッパーモードを有効にする。
3. 「パッケージ化されていない拡張機能を読み込む」を押す。
4. 展開した YTDExtender フォルダを指定して開く。
5. 導入完了です。
